杨辉三角

题目

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。
在杨辉三角中，每个数是它左上方和右上方的数的和。
示例:

输入: 5
输出:
[
[1],
[1,1],
[1,2,1],
[1,3,3,1],
[1,4,6,4,1]
]

思路

如果能够知道一行杨辉三角，我们就可以根据每对相邻的值轻松地计算出它的下一行。
首先，我们会生成整个 triangle 列表，三角形的每一行都以子列表的形式存储。然后，我们会检查行数为 0 的特殊情况，否则我们会返回 [1]。如果 numRows > 0，那么我们用 [1] 作为第一行来初始化 triangle with [1]，并按如下方式继续填充：
triangle[i] = triangle[pre-1] + triangle[pre+1]。理解为：每一个子列表的元素，除头尾元素外，都由上一层子列表元素的相邻两个元素的和。
三部曲：一初始化List;二添加首尾元素；三中间遍历添加每个和。结尾莫忘加进大列表。
参考LeetCode编程题

方法

动态规划

class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> allList = new ArrayList<List<Integer>>();
        //首先进行非法校验
        if(numRows < 0 || numRows == 0)
            return allList;
        //第一个元素一直为1（第一行）
        allList.add(new ArrayList<Integer>());    
        allList.get(0).add(1);
        for (int row=1; row<numRows;row++){
            List<Integer> list = new ArrayList<Integer>();
            List<Integer> preList = allList.get(row-1);
            //第一个元素一直为1（第row行）
            list.add(1);
            //每个子列表的元素，除头尾元素外，都由上一层子列表元素的相邻两个元素的和。
            //这里的列表大小也可以是row,理解为第row行有row个元素。这里理解添加的个数为上一行的元素个数
            for(int i=1;i<preList.size();i++)
                list.add(preList.get(i-1) + preList.get(i)); 
            //最后一个元素一直为1（第row行）
            list.add(1);
            allList.add(list);
        }
        return allList;
    }
}
